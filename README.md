# 💈 Barbearia Premium

> Landing page profissional e de alta conversão para barbearias, desenvolvida com foco total na experiência do usuário (UX) e facilidade de agendamento.

## 🚀 Objetivo
O objetivo principal deste projeto foi desenvolver uma interface institucional com apelo "Premium" e esteticamente refinada (estilo Dark mode), permitindo que os clientes consultem serviços, horários e entrem em contato de forma rápida. O projeto foi estruturado pensando em escalabilidade para se tornar um produto comercial real no futuro.

---

## 🛠️ Tecnologias e Conceitos
A aplicação foi construída do zero utilizando conceitos modernos de arquitetura de software e design de interface:

* **React**: Framework utilizado para construir uma interface modular, rápida e baseada em componentes reutilizáveis.
* **CSS Modules**: Abordagem que garante a estilização isolada de cada componente, mantendo a folha de estilos limpa e evitando conflitos globais de classes.
* **JavaScript (ES6+)**: Lógica para interatividades essenciais, como abertura de menus e manipulação de estados da interface.
* **Mobile First**: O desenvolvimento foi iniciado pensando estritamente na experiência em smartphones e expandido para desktops, garantindo uma responsividade impecável.
* **UX/UI Design**: Escolha de tipografia temática, espaçamentos generosos e botões de ação (CTAs) estratégicos para guiar a navegação do usuário.

---

## 🏗️ Estrutura da Aplicação
A arquitetura da página foi dividida em seções modulares para facilitar futuras manutenções e atualizações:

* **Header**: Menu de navegação responsivo (incluindo menu mobile hambúrguer).
* **Hero Section**: Banner principal de impacto com chamada para ação rápida.
* **About**: Seção institucional contando a história e o conceito da barbearia.
* **Services**: Vitrine de serviços com preços e descrições estilizadas.
* **Contact**: Informações de localização, horários e canais de atendimento.
* **WhatsApp Fix**: Botão flutuante fixo para conversão imediata e contato direto.
* **Footer**: Rodapé com direitos autorais e links rápidos para redes sociais.

## 🔄 Como Rodar o Projeto

Para executar este projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Herdes-s/Barbearia
   ```
   
2. **Acesse a pasta do projeto:**
    ```bash
    cd Barbearia
    ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```
   
4. **Inicie o servidor de desenvolvimento:**
    ```bash
    npm start
    ```

## 🧠 Evolução do Projeto (V1 vs V2)
* **O Desafio:** Na primeira versão (V1), o projeto cumpria o papel básico de exibir os dados, mas carecia de imersão visual e tinha limitações de navegação em telas menores.
* **A Solução (Refatoração V2):** Transformei a aplicação aplicando melhorias críticas de UX. Desenvolvi um **menu mobile interativo**, adicionei **efeitos de hover sutis**, integrei o **botão dinâmico do WhatsApp**, apliquei **animações de transição com CSS** e adotei uma tipografia muito mais temática e alinhada ao ambiente de uma barbearia premium.
